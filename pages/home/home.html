Datetime Range Picker

An angular directive for pick a range of dates and time (schedule). It requires angular ui.bootstrap so make sure you have it properly installed in your project

to time: {{myDatetimeRange.time.to | rgTime}} | {{myDatetimeRange.time.to | rgTime:true}}

How to install

Add the following files to your project:

<script src="range-picker.js"></script>

For now it only supports SASS styling, so import the range-picker.sass file to your sass project

Add rgkevin.datetimeRangePicker module as dependency in your angular app.

angular.module('yourApp', ['rgkevin.datetimeRangePicker']);

Install with bower

$ bower install datetimeRangePicker --save

Documentation

Directive 

<div rg-range-picker="myDatetimeRange" labels="myDatetimeLabels"></div>

where myDatetimeRange is a $scope.model object:

{
    date: {
        from: new Date(), // start date ( Date object )
        to: new Date() // end date ( Date object )
    },
    time: {
        from: 480, // default start time (in minutes)
        to: 1020, // default end time (in minutes)
        step: 15, // step width
        minRange: 15, // min range
        hours24: false // true = 00:00:00 | false = 00:00 am/pm
    }
}

and myDatetimeLabels is an $scope.model object:

{
    date: {
        from: 'Start date',
        to: 'End date'
    }
}

Directive with toggle

Yo make your this directive collapsible, just add the property collapse to the directive:

<div rg-range-picker="myDatetimeRange" collapse="collapseDirective" labels="myDatetimeLabels"></div>

In your button add ng-click="collapseDirective = !collapseDirective"

<button ng-click="collapseDirective = !collapseDirective">Toggle directive</button>
